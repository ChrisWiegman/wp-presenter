WP Presenter Theme
====================

A WordPress theme based on reveal.js.

The default keyboard shortcuts are:

Up, Down, Left, Right: Navigation
f: Full-screen
s: Show slide notes
o: Toggle overview
. (Period or b: Turn screen black
Esc: Escape from full-screen, or toggle overview
